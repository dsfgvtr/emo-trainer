# 🎓 Тренажёр ЭМО — Тестовое задание Web-программист

Интерактивный тренажёр для образовательной платформы **ЭМО** — многофункционального приложения для людей с ментальными особенностями.

## 📋 Описание

Тренажёр развивает навыки распознавания изображений и письменной речи. Пользователь видит изображение и должен ввести правильное название.

## 🚀 Функционал

### Основной
- ✅ 5 страниц вопросов с изображениями
- ✅ Проверка ответов в реальном времени
- ✅ Таблица результатов в конце
- ✅ Перезапуск тренажёра

### Дополнительный (*)
- ✅ Рандомизация вопросов (Fisher-Yates)
- ✅ Сохранение прогресса (localStorage)
- ✅ Уведомления о результатах
- ✅ Адаптивность под смартфоны
- ✅ Доступность (ARIA, keyboard navigation)
