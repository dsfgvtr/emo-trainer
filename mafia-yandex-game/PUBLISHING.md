## ✅ Что уже сделано

1. **Интеграция SDK Яндекс Игр** - полностью настроена в `src/yandex-sdk.js`
2. **Структура проекта** - готова к загрузке
3. **Манифест** - создан `yandex-manifest.json` с настройками
4. **Адаптивность** - игра работает на ПК и мобильных устройствах

## 📋 Чек-лист перед публикацией

### 1. Подготовьте графические активы

Создайте следующие файлы в папке `assets/`:

```
assets/
├── icon-512.png      # Иконка 512x512 px (обязательно)
├── favicon-16.png    # Фавикон 16x16 px
├── favicon-32.png    # Фавикон 32x32 px
├── favicon-64.png    # Фавикон 64x64 px
├── screenshot-1.png  # Скриншот 1920x1080 px (минимум 3 шт)
├── screenshot-2.png
├── screenshot-3.png
└── cover.jpg         # Обложка 1920x1080 px (рекомендуется)
```

**Требования к иконке:**
- Квадратное изображение
- Без текста (или минимальный текст)
- Яркая, запоминающаяся
- Формат PNG с прозрачным фоном или JPG

### 2. Протестируйте игру локально

```bash
cd /workspace/mafia-yandex-game
npm start
```

Откройте http://localhost:8080 и проверьте:
- [ ] Все кнопки работают
- [ ] Диалоги отображаются корректно
- [ ] Выбор влияет на сюжет
- [ ] Сохранения работают
- [ ] Адаптивность на разных экранах

### 3. Зарегистрируйтесь в консоли разработчика

1. Перейдите на https://games.yandex.ru/console/
2. Войдите через Яндекс ID
3. Нажмите "Добавить приложение"

### 4. Заполните информацию о приложении

**Основные поля:**
- Название: "Мафия: Тени Нью-Валлетто"
- Описание: (возьмите из README.md)
- Категория: Приключения / Ролевые
- Возраст: 18+
- Язык: Русский

**Загрузите файлы:**
- Исходный код (ZIP архив с содержимым папки `mafia-yandex-game`)
- Иконки
- Скриншоты

### 5. Настройки в консоли

**Черновик:**
- Ориентация: Портретная
- Полноэкранный режим: Нет
- Управление: Мышь/Касания

**Монетизация (опционально):**
- Реклама:Interstitial (между главами)
- Внутренние покупки: Нет

### 6. Модерация

После отправки на модерацию:
- Ожидайте 1-3 рабочих дня
- Проверьте почту для уведомлений
- При отказе исправьте замечания и отправьте повторно

cd /workspace/mafia-yandex-game

## 🐛 Решение проблем

### Игра не загружается в Яндекс Играх

**Проблема:** Ошибка CORS
**Решение:** Убедитесь, что все пути относительные (без `/` в начале)

**Проблема:** SDK не инициализируется
**Решение:** Проверьте подключение скрипта в `<head>`:
```html
<script src="https://yandex.ru/games/sdk/v2"></script>
```

### Реклама не показывается

**Причина:** Требуется модерация
**Решение:** После прохождения модерации реклама активируется автоматически

### Сохранения не работают

**Проверьте:**
1. Игрок авторизован в Яндексе
2. Вызывается `saveData()` после важных событий
3. Нет ошибок в консоли
