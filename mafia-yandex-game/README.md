# Мафия: Тени Нью-Валлетто (Mafia: Tenebrae)

Масштабная текстовая RPG-игра в стиле классической Мафии для платформы Яндекс Игры.

## 🎮 Особенности игры

- **Глубокий сюжет**: 3 главы с ветвящимися диалогами и несколькими концовками
- **Система репутации**: Ваши решения влияют на отношение семьи к вам
- **Ролевая система**: Выберите роль (Солдат, Капо, Консильери) - каждая даёт уникальные бонусы
- **Экономическая система**: Управление деньгами, подкуп, затраты
- **Сохранение прогресса**: Облачные сохранения через Яндекс SDK
- **Адаптивный дизайн**: Работает на ПК и мобильных устройствах

## 📖 Сюжет

1945 год. Вы возвращаетесь с войны в родной Нью-Валлетто. Город изменился - теперь им правит Семья Моретти. Но ирландцы из Бруклина нарушили перемирие, и начинается война...

Станьте частью криминальной империи, принимайте сложные решения, доказывайте свою преданность и поднимайтесь по иерархии!

## 🚀 Быстрый старт

### Локальный запуск

```bash
cd mafia-yandex-game
npm install
npm start
```

Игра откроется по адресу: http://localhost:8080

### Структура проекта

```
mafia-yandex-game/
├── index.html          # Основной HTML файл
├── css/
│   └── style.css       # Стили игры
├── src/
│   ├── main.js         # Точка входа
│   ├── yandex-sdk.js   # Интеграция с Яндекс Играми
│   ├── game-engine.js  # Игровой движок
│   └── story-data.js   # Данные сюжета и диалоги
```

## 🎯 Как играть

1. Нажмите "Начать игру" в главном меню
2. Создайте персонажа (имя и роль)
3. Читайте диалоги и делайте выбор
4. Следите за репутацией и деньгами
5. Проходите главы и открывайте новые части истории

## 🔧 Требования для публикации в Яндекс Играх

1. **Иконки**: 
   - 512x512 px (основная)
   - 16x16, 32x32, 64x64 px (favicon)

2. **Скриншоты**: Минимум 3 скриншота 1920x1080 px

3. **Описание**: Заполните в консоли разработчика Яндекс Игр

4. **SDK**: Уже интегрирован в `src/yandex-sdk.js`

## 📝 Расширение игры

### Добавление новых сцен

Откройте `src/story-data.js` и добавьте новую сцену:

```javascript
новая_сцена: {
    id: "новая_сцена",
    speaker: "Имя персонажа",
    text: "Текст диалога...",
    background: "название_фона",
    reputationChange: 10, // опционально
    choices: [
        { text: "Выбор 1", nextScene: "следующая_сцена" },
        { text: "Выбор 2", nextScene: "другая_сцена", reqRep: 70 } // требуется репутация
    ]
}
```

### Добавление фонов

В `game-engine.js` в функцию `updateBackground` добавьте новые цвета или URL изображений.
